# 页面/区块预览

### 前言

在一些场景下，设计器画布无法直观展示区块或页面的最终呈现效果。比如说，当页面中的区块与组件嵌套层级过多，或彼此间的交互逻辑复杂，我们将难以在设计画布上直观感知其静态布局以及复杂的动态交互行为，这时候就需要设计器的**页面/区块预览**能力来为开发提供帮助

- 国际化语言
- state状态响应式
- 表单验证提交
